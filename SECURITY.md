# Security Policy

This is an educational research repository, not an audited or supported production package.

Do not open a public issue for a vulnerability that could expose people, credentials, data, or systems. Contact the repository owner privately through the contact method on the maintainer's GitHub profile. Include the affected commit, reproduction, impact, and suggested mitigation. An acknowledgement is targeted within seven days, but no bounty or fixed remediation timeline is offered.

CIFAR-10, Python packages, notebooks, and model artifacts are third-party or executable inputs. Review their sources and terms, isolate untrusted notebooks, and do not place credentials in the repository.
